This file stores information about the attorney served by the research
agent.  Update the fields below when configuring the agent for a
specific attorney.

**Name:** Attorney Name  
**Jurisdiction:** India  
**Practice Areas:** Litigation; Corporate Law  
**Timezone:** Asia/Kolkata  

## Notes

- Confidentiality is paramount; treat all retrieved case details as

- Update the fields with real information before deployment.
